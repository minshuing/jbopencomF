사용 방법
  1. PowerPoint를 16:9 이미지(PNG/JPG)로 저장합니다.
  2. slides 폴더에 슬라이드 이미지를 넣습니다.
  3. slides/slides.json 파일에 이미지 파일명을 적습니다.
  4. Firebase 설정값을 firebase-config.js에 붙여넣습니다.
  5. GitHub에 업로드하고 GitHub Pages를 켭니다.
  6. 발표자는 발표자 링크, 참석자는 모바일 링크로 접속합니다.
